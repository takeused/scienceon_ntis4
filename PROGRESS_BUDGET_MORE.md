# 연구비 결과 목록 확장 기록

## 2026-07-13

- 대표 유사과제 표는 초기 3건을 우선 표시한다.
- 4건 이상이면 `더 보기` 버튼을 조건부로 렌더링한다.
- 추가 후보는 계산에서 제외하지 않고, 버튼 토글로 행만 펼치거나 접는다.
- `aria-expanded` 상태와 버튼 라벨·아이콘을 함께 갱신한다.
- 검증: 관련 테스트 포함 전체 25개 통과.

## 2026-07-13 홈 키워드 칩 동작 변경

- 하단 예시 키워드 클릭 시 즉시 ScienceON 검색하지 않고 검색창에 값을 입력한다.
- 검색창에 포커스하고 텍스트를 선택해 사용자가 바로 수정할 수 있게 했다.
- 검색은 통합 검색 버튼 또는 Enter 입력으로 확정한다.
- 검증: 전체 테스트 27개 통과.
