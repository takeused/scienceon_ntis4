# 연구–IP 전환 공백 분석 로직 심층 검토 및 개선 기록

## 2026-07-13 결론

분석 결과는 사업화 성공확률이 아니라 후속 검토 우선순위로 유지한다. 이번 개선은 공백 점수의 수학적 기준, 후보군 비교의 데이터 자격, 검색 완화와 API 오류의 신뢰도 반영, 추세 상태, 다양성 제거, 상세 리포트 해석을 일관되게 만드는 데 초점을 맞췄다.

## 확인된 핵심 문제와 개선

1. **공백 공식의 기준점 오류**
   - 기존 `log(논문) / (log(논문) + log(특허))`는 논문과 특허가 같은 규모여도 공백 50점을 만들었다.
   - `1 - (log(특허) / log(논문))`으로 변경해 같은 규모는 공백 0, 특허강도가 낮을수록 공백 신호가 커지게 했다.
   - 특허 0건은 검색 실패·분류 차이 가능성이 있어 기존처럼 70점 상한을 유지한다.

2. **누락·오류 데이터가 정상으로 처리되는 문제**
   - 지표 객체가 없을 때 정상으로 간주하던 조건을 제거했다.
   - 논문·특허 핵심 지표가 없거나 오류이면 정식 순위에서 제외한다.

3. **후보군 중앙값 오염**
   - API 오류 후보와 논문 5건 미만 후보가 상대 특허강도 중앙값을 왜곡할 수 있었다.
   - 핵심 지표가 정상이고 논문 5건 이상인 후보만 포함하며, 유효 후보가 3개 미만이면 후보군 보정을 사용하지 않는다.

4. **검색 완화의 범위 드리프트**
   - 완화 검색식의 순서와 관계없이 동일한 15% 페널티만 적용됐다.
   - 원 검색어에서 멀어진 단계가 깊을수록 데이터 신뢰도 페널티가 커지도록 변경했다.
   - 논문과 특허가 모두 정상 조회된 검색식만 canonical 후보로 우선 채택한다.

5. **요청 폭주가 데이터 오류로 전환되는 문제**
   - 후보 6개를 동시에 검증하면 ScienceON·NTIS rate limit이 후보 점수 차이로 오인될 수 있었다.
   - 후보 동시 검증을 2개로 제한하고, 후보 내부 검색식은 순차 검증한다.

6. **빈 추세가 정상 추세로 처리되는 문제**
   - 최근 4개 연도가 모두 0이어도 정상 상태와 중립 성장으로 해석될 수 있었다.
   - `ok/partial/error/empty`를 분리하고, empty·partial·error는 성장 모멘텀을 확정하지 않는다.
   - 작은 모수 폭증을 완화하는 20건 사전값 기반 수축 성장률과 원 성장률을 분리해 기록한다.

7. **다양성 제거 후 유사 후보 재삽입**
   - Top 3 숫자를 채우기 위해 다양성 규칙으로 제외한 후보를 다시 넣고 있었다.
   - 서로 다른 유효 후보가 부족하면 Top 1~2만 반환하도록 변경했다.

8. **순위 카드와 상세 리포트 공식 불일치**
   - 상세 화면이 과거 공백 공식을 다시 계산해 순위 카드와 다른 결과를 보일 수 있었다.
   - 상세 화면은 순위 산출에 사용한 동일 `gapSignal`을 재사용한다.

9. **근거 점수와 데이터 연결 범위 혼합**
   - 미연결 특허패밀리·시장·TRL 데이터가 부정 근거 0점처럼 취급됐다.
   - 관측 가능한 근거로 점수를 정규화하고 `근거 범위`를 별도 표시한다.
   - 보고서 1건을 만점으로 보던 이진 지표를 로그 규모 지표로 변경했다.

10. **상위 표본에 기반한 과도한 사업화 단정**
    - 상위 특허 몇 건만으로 출원인 편중, 선점 기회, 원천특허 출원을 제안했다.
    - 상위 결과는 표본임을 명시하고, 고정 별점과 단정적 시나리오를 `후속 검증 경로`로 교체했다.
    - 특허성·FTO·패밀리·IPC/CPC·수요·TRL 검증 전에는 출원·투자 결론을 내리지 않는다.

## 현재 산식

- 공백 매력도 = 연구 기반 35% + IP 공백 신호 40% + 연구 모멘텀 25%
- IP 공백 신호 = 직접 로그 특허강도 결핍 70% + 신뢰 가능한 후보군 상대 신호 30%
- 내부 순위 = 공백 매력도 × 데이터 신뢰도
- 전환·실행 근거는 순위와 분리하고, 점수와 근거 범위를 함께 표시한다.
- 정식 순위 후보 = 논문 20건 이상 + 논문·특허 핵심 조회 정상 + 데이터 신뢰도 60점 이상
- 논문 5~19건 후보는 성장·신뢰도 조건을 만족해도 정식 순위가 아니라 초기 탐색 후보로 분리한다.

## 남은 구조적 한계

- ScienceON 검색 건수는 특허 패밀리·유효권리·법적 상태를 반영하지 않는다.
- 후보군 중앙값은 현재 분석에서 생성된 6개 안팎 후보의 상대 기준이며 산업 전체 기준이 아니다.
- AI가 생성한 검색어의 의미적 동등성은 완화 단계 페널티로만 근사하며, IPC/CPC 기반 검증은 아직 없다.
- NTIS·보고서 건수는 투자 규모·성과 품질·과제 중복성을 직접 의미하지 않는다.
- 시장 규모, 수요기관 검증, TRL, FTO가 연결되기 전에는 사업화 결론으로 사용할 수 없다.
